Bunny Money:
Соберите 50 Treasure Sack Pieces охотясь на монстров.
Когда вы сдадите 50 Treasure Sack Pieces одному из Bunny Money эвент менеджеров, вы получите один из шести различных Treasure Sacks, откройте его (двойным щелчком) для того чтобы увидеть что вы выиграли!
Большинство Treasure Sacks содержат один предмет, но некоторые содержат несколько.
