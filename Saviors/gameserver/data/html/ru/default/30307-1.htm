Излишне говорить, что для того, чтобы сделать Dualsword, сначала необходимо два меча.
Это может быть обычное оружие, Common оружие или Masterwork оружие.
Обычное оружие можно использовать только с обычным, Common оружие можно использовать только с Common оружием, и Masterwork оружие только с Masterwork оружием. Кроме того, Masterwork Dualsword могут быть созданы из мечей только выше B grade.
